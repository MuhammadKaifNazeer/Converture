## Getting Started

To get started with Converture, follow these steps:

1. **Clone the repository:**

```bash
git clone https://github.com/yourusername/converture.git
```

2. **Install the dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```
